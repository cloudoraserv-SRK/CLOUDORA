Frequently Asked Questions

Do you create original music or sound effects?

Yes, we can either source licensed, royalty-free soundtracks or work with audio designers to create custom music and sound effects as part of a package.

What file formats do you deliver?

For logos, we deliver vector files (SVG, AI, EPS) and high-resolution raster files (PNG, JPG). For videos, we provide high-quality MP4 files in various aspect ratios (16:9, 9:16, 1:1) as needed.

How many revisions are included?

Each package includes a set number of revisions to ensure you are completely satisfied with the final product. Additional revisions can be added as an a la carte service.
